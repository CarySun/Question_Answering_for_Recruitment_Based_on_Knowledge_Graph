### 二. 知识图谱构建模块
- step1: 职位->技能, 工作描述, 公司, 属性(工作地点, 工作经验, 工资)
- step2: 公司->公司评价标签, 属性(类型, 人数, 官网, 描述, 注册资本, 地址, 工资)
- step3: 技能节点进行NER, 提取关键词
- step4: 技能->百度百科信息, 推荐资料节点(视频, 书籍, 网站)
- step5: 工作描述进行断句以及主谓分析
